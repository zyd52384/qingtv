# 轻看TV

**把官网电视直播装到智能电视上，遥控器一键换台。**

---

## 这是啥

轻看TV 是一个简约的 Android TV 应用，用 WebView 加载电视台官网直播页面。

**不用装任何中间件、不用配置任何源、不用折腾任何设置。**

> 装好 APK → 打开 → 选台 → 看

## 频道列表

| 频道 | 按键 |
|------|------|
| CCTV-1 ~ CCTV-9 | 数字键 1 ~ 9 |
| CCTV-10 ~ CCTV-17 | 按数字键切换 |
| CCTV-新闻 | 数字键 0 |
| 其他频道 | 方向键选择 |

## 版本说明

### 免费版（公开下载）

- ✅ 19个CCTV频道
- ✅ 遥控器数字键换台
- ✅ 方向键浏览选择
- ❌ 底部有免费版标识
- ❌ **不支持全屏播放**（页面内播放）

### 社群版（Pro）

- ✅ 全部频道
- ✅ 遥控器数字键换台
- ✅ **全屏沉浸播放**
- ✅ 频道自定义排序
- ✅ 频道源定期更新
- ✅ 无任何标识

## 法律声明

```
本应用仅展示各电视台官方网站的公开直播页面。
所有内容来自原网站，版权归各电视台所有。

```

## 技术栈

- Kotlin + Android Jetpack
- WebView + X5内核支持
- GitHub Actions 自动编译APK
- 双Flavor（free/pro）构建

本应用仅供学习交流使用，基于 WebView 技术的电视直播应用示例。请勿用于商业用途或二次打包盈利。
